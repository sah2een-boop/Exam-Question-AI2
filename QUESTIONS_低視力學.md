# 低視力學 - 示範題庫

請將下方表格內容複製並貼上至您的 Google Sheet「低視力學」工作表中。

## 題目列表

### 題目 1
**題目：** 根據 WHO 定義，低視力 (Low Vision) 是指矯正後視力在什麼範圍？

| 選項 | 內容 |
|------|------|
| A | 低於 0.3 (6/18) |
| B | 低於 0.5 (6/12) |
| C | 低於 0.8 (6/7.5) |
| D | 低於 1.0 (6/6) |

**正確答案：** A - 低於 0.3 (6/18)

---

### 題目 2
**題目：** 法定盲 (Legal Blindness) 在台灣的定義是矯正後視力低於多少？

| 選項 | 內容 |
|------|------|
| A | 0.1 (6/60) |
| B | 0.05 (3/60) |
| C | 0.01 (1/60) |
| D | 0.3 (6/18) |

**正確答案：** B - 0.05 (3/60)

---

### 題目 3
**題目：** 低視力輔具中，放大鏡主要幫助什麼類型的視力問題？

| 選項 | 內容 |
|------|------|
| A | 視野缺損 |
| B | 中心視力不良 |
| C | 夜盲症 |
| D | 色盲 |

**正確答案：** B - 中心視力不良

---

### 題目 4
**題目：** 黃斑部病變 (Macular Degeneration) 患者主要會出現什麼視力問題？

| 選項 | 內容 |
|------|------|
| A | 周邊視野缺損 |
| B | 中心視力喪失 |
| C | 夜間視力下降 |
| D | 色覺異常 |

**正確答案：** B - 中心視力喪失

---

### 題目 5
**題目：** 視網膜色素病變 (Retinitis Pigmentosa) 患者典型的視野變化是？

| 選項 | 內容 |
|------|------|
| A | 中心暗點 |
| B | 管狀視野 (周邊視野喪失) |
| C | 半側視野缺損 |
| D | 視野正常 |

**正確答案：** B - 管狀視野 (周邊視野喪失)

---

### 題目 6
**題目：** 對於低視力患者，提高照明度主要能改善什麼？

| 選項 | 內容 |
|------|------|
| A | 視野範圍 |
| B | 對比敏感度和視力 |
| C | 色覺 |
| D | 眼壓 |

**正確答案：** B - 對比敏感度和視力

---

### 題目 7
**題目：** 電子放大鏡 (CCTV) 相較於光學放大鏡的優點是？

| 選項 | 內容 |
|------|------|
| A | 價格較便宜 |
| B | 可調整對比度和放大倍率 |
| C | 攜帶更方便 |
| D | 不需電源 |

**正確答案：** B - 可調整對比度和放大倍率

---

### 題目 8
**題目：** 低視力復健的主要目標是什麼？

| 選項 | 內容 |
|------|------|
| A | 完全恢復視力 |
| B | 提升患者生活品質和獨立性 |
| C | 治癒眼疾 |
| D | 預防其他疾病 |

**正確答案：** B - 提升患者生活品質和獨立性

---

### 題目 9
**題目：** 偏心注視訓練 (Eccentric Viewing Training) 主要用於哪類患者？

| 選項 | 內容 |
|------|------|
| A | 青光眼 |
| B | 黃斑部病變 |
| C | 白內障 |
| D | 近視 |

**正確答案：** B - 黃斑部病變

---

### 題目 10
**題目：** 對於閱讀困難的低視力患者，除了放大鏡外，還可以建議什麼輔具？

| 選項 | 內容 |
|------|------|
| A | 太陽眼鏡 |
| B | 有聲書或語音閱讀器 |
| C | 隱形眼鏡 |
| D | 眼藥水 |

**正確答案：** B - 有聲書或語音閱讀器
