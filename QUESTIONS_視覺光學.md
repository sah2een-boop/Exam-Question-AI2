# 視覺光學 - 示範題庫

請將下方表格內容複製並貼上至您的 Google Sheet「視覺光學」工作表中。

## 題目列表

### 題目 1
**題目：** 近視眼的光學特徵是什麼？

| 選項 | 內容 |
|------|------|
| A | 光線聚焦在視網膜前方 |
| B | 光線聚焦在視網膜後方 |
| C | 光線無法聚焦 |
| D | 光線聚焦在黃斑部 |

**正確答案：** A - 光線聚焦在視網膜前方

---

### 題目 2
**題目：** 矯正近視應使用何種鏡片？

| 選項 | 內容 |
|------|------|
| A | 凸透鏡 (正鏡片) |
| B | 凹透鏡 (負鏡片) |
| C | 柱狀鏡片 |
| D | 稜鏡 |

**正確答案：** B - 凹透鏡 (負鏡片)

---

### 題目 3
**題目：** 散光的主要成因是什麼？

| 選項 | 內容 |
|------|------|
| A | 眼軸過長 |
| B | 角膜或水晶體曲率不均勻 |
| C | 水晶體混濁 |
| D | 視網膜剝離 |

**正確答案：** B - 角膜或水晶體曲率不均勻

---

### 題目 4
**題目：** 老花眼 (Presbyopia) 的主要原因是？

| 選項 | 內容 |
|------|------|
| A | 角膜變形 |
| B | 水晶體彈性下降 |
| C | 視網膜退化 |
| D | 眼軸變短 |

**正確答案：** B - 水晶體彈性下降

---

### 題目 5
**題目：** 屈光度 (Diopter) 的單位符號是？

| 選項 | 內容 |
|------|------|
| A | m |
| B | D |
| C | cm |
| D | mm |

**正確答案：** B - D

---

### 題目 6
**題目：** 一個焦距為 50 公分的透鏡，其屈光度為多少？

| 選項 | 內容 |
|------|------|
| A | +0.5D |
| B | +2.0D |
| C | +5.0D |
| D | +50D |

**正確答案：** B - +2.0D

---

### 題目 7
**題目：** 遠視眼配戴眼鏡後，影像會呈現什麼變化？

| 選項 | 內容 |
|------|------|
| A | 影像放大 |
| B | 影像縮小 |
| C | 影像不變 |
| D | 影像模糊 |

**正確答案：** A - 影像放大

---

### 題目 8
**題目：** 稜鏡的主要功能是什麼？

| 選項 | 內容 |
|------|------|
| A | 放大影像 |
| B | 改變光線方向 |
| C | 增加對比度 |
| D | 過濾紫外線 |

**正確答案：** B - 改變光線方向

---

### 題目 9
**題目：** 正常眼睛的遠點 (Far Point) 位於何處？

| 選項 | 內容 |
|------|------|
| A | 無限遠 |
| B | 眼前 25 公分 |
| C | 眼前 1 公尺 |
| D | 視網膜上 |

**正確答案：** A - 無限遠

---

### 題目 10
**題目：** 雙光鏡片 (Bifocal Lens) 主要用於矯正什麼問題？

| 選項 | 內容 |
|------|------|
| A | 近視 |
| B | 遠視 |
| C | 散光 |
| D | 老花眼合併近視或遠視 |

**正確答案：** D - 老花眼合併近視或遠視
