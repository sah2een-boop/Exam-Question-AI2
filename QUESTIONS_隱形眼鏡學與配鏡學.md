# 隱形眼鏡學與配鏡學 - 示範題庫

請將下方表格內容複製並貼上至您的 Google Sheet「隱形眼鏡學與配鏡學」工作表中。

## 題目列表

### 題目 1
**題目：** 軟式隱形眼鏡的主要材質是什麼？

| 選項 | 內容 |
|------|------|
| A | 玻璃 |
| B | 水膠 (Hydrogel) |
| C | 金屬 |
| D | 陶瓷 |

**正確答案：** B - 水膠 (Hydrogel)

---

### 題目 2
**題目：** 硬式隱形眼鏡 (RGP) 相較於軟式隱形眼鏡的優點是？

| 選項 | 內容 |
|------|------|
| A | 配戴更舒適 |
| B | 透氧性更好 |
| C | 價格更便宜 |
| D | 不需清潔保養 |

**正確答案：** B - 透氧性更好

---

### 題目 3
**題目：** 隱形眼鏡的基弧 (Base Curve, BC) 是指什麼？

| 選項 | 內容 |
|------|------|
| A | 鏡片的直徑 |
| B | 鏡片內表面的曲率半徑 |
| C | 鏡片的厚度 |
| D | 鏡片的度數 |

**正確答案：** B - 鏡片內表面的曲率半徑

---

### 題目 4
**題目：** 配戴隱形眼鏡時，出現「3-9 點鐘染色」通常表示什麼問題？

| 選項 | 內容 |
|------|------|
| A | 鏡片過緊 |
| B | 鏡片過鬆導致邊緣乾燥 |
| C | 度數不正確 |
| D | 鏡片破損 |

**正確答案：** B - 鏡片過鬆導致邊緣乾燥

---

### 題目 5
**題目：** 角膜塑型片 (Orthokeratology) 主要在什麼時候配戴？

| 選項 | 內容 |
|------|------|
| A | 白天 |
| B | 夜間睡眠時 |
| C | 運動時 |
| D | 游泳時 |

**正確答案：** B - 夜間睡眠時

---

### 題目 6
**題目：** 眼鏡鏡片的光學中心應對準什麼位置？

| 選項 | 內容 |
|------|------|
| A | 眉心 |
| B | 瞳孔中心 |
| C | 鼻樑 |
| D | 眼角 |

**正確答案：** B - 瞳孔中心

---

### 題目 7
**題目：** 漸進多焦點鏡片 (Progressive Lens) 的主要特點是什麼？

| 選項 | 內容 |
|------|------|
| A | 只有一個焦點 |
| B | 有明顯分界線 |
| C | 焦點漸進變化無分界線 |
| D | 只能看遠 |

**正確答案：** C - 焦點漸進變化無分界線

---

### 題目 8
**題目：** 高度近視患者配鏡時，建議選用什麼材質的鏡片？

| 選項 | 內容 |
|------|------|
| A | 玻璃鏡片 |
| B | 高折射率鏡片 |
| C | 低折射率鏡片 |
| D | 塑膠鏡片 |

**正確答案：** B - 高折射率鏡片

---

### 題目 9
**題目：** 抗藍光鏡片主要用於防護什麼？

| 選項 | 內容 |
|------|------|
| A | 紫外線 |
| B | 可見光中的藍光 |
| C | 紅外線 |
| D | X 光 |

**正確答案：** B - 可見光中的藍光

---

### 題目 10
**題目：** 配戴隱形眼鏡時，每日配戴時間不應超過多少小時？

| 選項 | 內容 |
|------|------|
| A | 6 小時 |
| B | 8 小時 |
| C | 10 小時 |
| D | 12 小時 |

**正確答案：** B - 8 小時
