# 眼球解剖生理學與倫理法規 - 示範題庫

## 題目列表

### 題目 1
**題目：** 眼球最外層堅韌的白色保護膜稱為什麼？

| 選項 | 內容 |
|------|------|
| A | 角膜 (Cornea) |
| B | 鞏膜 (Sclera) |
| C | 視網膜 (Retina) |
| D | 脈絡膜 (Choroid) |

**正確答案：** B - 鞏膜 (Sclera)

---

### 題目 2
**題目：** 眼睛中負責將光線聚焦在視網膜上的透明結構是？

| 選項 | 內容 |
|------|------|
| A | 虹膜 (Iris) |
| B | 玻璃體 (Vitreous Humor) |
| C | 水晶體 (Lens) |
| D | 瞳孔 (Pupil) |

**正確答案：** C - 水晶體 (Lens)

---

### 題目 3
**題目：** 控制進入眼睛光線量的結構是？

| 選項 | 內容 |
|------|------|
| A | 視神經 (Optic Nerve) |
| B | 角膜 (Cornea) |
| C | 虹膜 (Iris) |
| D | 鞏膜 (Sclera) |

**正確答案：** C - 虹膜 (Iris)

---

### 題目 4
**題目：** 視網膜上負責中央視覺和分辨顏色的區域稱為？

| 選項 | 內容 |
|------|------|
| A | 視盤 (Optic Disc) |
| B | 黃斑部 (Macula) |
| C | 睫狀體 (Ciliary Body) |
| D | 前房 (Anterior Chamber) |

**正確答案：** B - 黃斑部 (Macula)

---

### 題目 5
**題目：** 眼球內部填充的透明膠狀物質，幫助維持眼球形狀的是？

| 選項 | 內容 |
|------|------|
| A | 房水 (Aqueous Humor) |
| B | 淚液 (Tears) |
| C | 玻璃體 (Vitreous Humor) |
| D | 血漿 (Plasma) |

**正確答案：** C - 玻璃體 (Vitreous Humor)

---

### 題目 6
**題目：** 對光線極為敏感，負責夜間視覺的感光細胞是？

| 選項 | 內容 |
|------|------|
| A | 錐狀細胞 (Cones) |
| B | 桿狀細胞 (Rods) |
| C | 雙極細胞 (Bipolar Cells) |
| D | 神經節細胞 (Ganglion Cells) |

**正確答案：** B - 桿狀細胞 (Rods)

---

### 題目 7
**題目：** 主要負責眼睛光線折射（佔總屈光力約 2/3）的結構是？

| 選項 | 內容 |
|------|------|
| A | 水晶體 (Lens) |
| B | 角膜 (Cornea) |
| C | 玻璃體 (Vitreous Humor) |
| D | 房水 (Aqueous Humor) |

**正確答案：** B - 角膜 (Cornea)

---

### 題目 8
**題目：** 負責將視網膜接收到的視覺訊號傳送到大腦的結構是？

| 選項 | 內容 |
|------|------|
| A | 視神經 (Optic Nerve) |
| B | 視交叉 (Optic Chiasm) |
| C | 外膝狀體 (LGN) |
| D | 視放射 (Optic Radiations) |

**正確答案：** A - 視神經 (Optic Nerve)

---

### 題目 9
**題目：** 位於虹膜和角膜之間的空間充滿了什麼液體？

| 選項 | 內容 |
|------|------|
| A | 玻璃體 (Vitreous Humor) |
| B | 淋巴液 (Lymph) |
| C | 房水 (Aqueous Humor) |
| D | 生理食鹽水 (Saline) |

**正確答案：** C - 房水 (Aqueous Humor)

---

### 題目 10
**題目：** 視網膜上沒有感光細胞，被稱為「盲點」的區域是？

| 選項 | 內容 |
|------|------|
| A | 黃斑部 (Macula) |
| B | 視凹 (Fovea) |
| C | 視盤 (Optic Disc) |
| D | 鋸齒緣 (Ora Serrata) |

**正確答案：** C - 視盤 (Optic Disc)
