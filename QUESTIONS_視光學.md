# 視光學 - 示範題庫

請將下方表格內容複製並貼上至您的 Google Sheet「視光學」工作表中。

## 題目列表

### 題目 1
**題目：** 進行驗光時，第一步應該先檢查什麼？

| 選項 | 內容 |
|------|------|
| A | 眼壓 |
| B | 視力 |
| C | 眼位 |
| D | 色覺 |

**正確答案：** B - 視力

---

### 題目 2
**題目：** 自覺式驗光法 (Subjective Refraction) 主要依賴什麼？

| 選項 | 內容 |
|------|------|
| A | 儀器測量數據 |
| B | 患者的主觀回應 |
| C | 醫師的經驗判斷 |
| D | 電腦自動分析 |

**正確答案：** B - 患者的主觀回應

---

### 題目 3
**題目：** 他覺式驗光法 (Objective Refraction) 常用的儀器是？

| 選項 | 內容 |
|------|------|
| A | 裂隙燈 |
| B | 視網膜鏡 (Retinoscope) |
| C | 眼壓計 |
| D | 視野計 |

**正確答案：** B - 視網膜鏡 (Retinoscope)

---

### 題目 4
**題目：** 紅綠測試 (Duochrome Test) 主要用於檢查什麼？

| 選項 | 內容 |
|------|------|
| A | 色盲 |
| B | 散光軸度 |
| C | 球面度數的精確度 |
| D | 眼位異常 |

**正確答案：** C - 球面度數的精確度

---

### 題目 5
**題目：** 交叉圓柱鏡 (Jackson Cross Cylinder) 主要用於精確測量什麼？

| 選項 | 內容 |
|------|------|
| A | 球面度數 |
| B | 散光度數和軸度 |
| C | 稜鏡度數 |
| D | 瞳孔距離 |

**正確答案：** B - 散光度數和軸度

---

### 題目 6
**題目：** 正常成人的瞳孔距離 (PD) 平均約為多少？

| 選項 | 內容 |
|------|------|
| A | 50-55 mm |
| B | 60-65 mm |
| C | 70-75 mm |
| D | 80-85 mm |

**正確答案：** B - 60-65 mm

---

### 題目 7
**題目：** 進行散瞳驗光時，常用的散瞳劑是？

| 選項 | 內容 |
|------|------|
| A | Atropine |
| B | Pilocarpine |
| C | Timolol |
| D | Latanoprost |

**正確答案：** A - Atropine

---

### 題目 8
**題目：** 雙眼視功能檢查中，Worth 四點燈測試主要檢查什麼？

| 選項 | 內容 |
|------|------|
| A | 視力 |
| B | 融像功能 |
| C | 色覺 |
| D | 視野 |

**正確答案：** B - 融像功能

---

### 題目 9
**題目：** 隱斜視 (Phoria) 與顯斜視 (Tropia) 的主要差異是？

| 選項 | 內容 |
|------|------|
| A | 隱斜視可自行矯正，顯斜視無法 |
| B | 隱斜視較嚴重 |
| C | 顯斜視可自行矯正 |
| D | 兩者相同 |

**正確答案：** A - 隱斜視可自行矯正，顯斜視無法

---

### 題目 10
**題目：** 調節幅度 (Amplitude of Accommodation) 會隨著年齡如何變化？

| 選項 | 內容 |
|------|------|
| A | 逐漸增加 |
| B | 逐漸減少 |
| C | 保持不變 |
| D | 先增後減 |

**正確答案：** B - 逐漸減少
